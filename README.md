# Start Here

### 1. Create a virtual enviroment:

```powershell
python -m venv .venv
```

### 2. Active your venv:

Windows:
```powershell
.venv\Scripts\activate
```

Linux or MacOS:
```powershell
source .venv/bin/activate
```

### 3. Install requirements.txt:

```powershell
pip install -r requirements.txt
```
